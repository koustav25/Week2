```bash
git clone ...
```

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

Install the requirements.

```bash
pip install -r requirements.txt
```

Run the Jypyter Notebook.

```bash
jupyter notebook w2_understandingUnfamiliarSystems.ipynb
```
